A portfolio of 9+ WordPress themes customized and refined for various clients. Each theme was either adapted from existing designs or built from Figma mockups, with comprehensive customization using Advanced Custom Fields (ACF) for flexible content management.

## Customized Themes Portfolio

- **[Booking Heads](https://bookingheads.com/)** - Travel/booking theme with reservation system
- **[IP2FY](https://ip2fy.com/)** - Technology services theme
- **[Uptime Shepherd](https://uptimeshepherd.com/)** - Monitoring/infrastructure services theme
- **[Pixel Pylon](https://pixelpylon.com/)** - Digital agency theme
- **[Skate.ge](https://skate.ge/)** - Sports/skateboarding community theme
- **[Wildcard.ge](https://wildcard.ge/)** - Entertainment/events theme
- **[Paracounter](https://paracounter.com/)** - Niche service industry theme
- **[Inbox Signature](https://inboxsignature.com/)** - Communication/email services theme
- **[A11Y Widget](https://a11ywidget.com/)** - Accessibility tools theme

## Customization & Development

- Theme selection and adaptation for client needs
- Custom field setup using Advanced Custom Fields (ACF)
- Plugin integration and customization
- Responsive design refinement for all devices
- Custom post types and taxonomies configuration
- JavaScript enhancements and interactions
- Performance optimization and caching
- SEO-friendly structure and metadata implementation

## Technologies Used

- CMS: WordPress with multisite capability
- Language: PHP, JavaScript
- Fields Management: Advanced Custom Fields (ACF)
- Design: Figma-to-theme adaptation and customization
- Styling: CSS3, SCSS
- Database: MySQL
- Hosting: Client-managed or WordPress hosting platforms (WP Engine, Kinsta, etc.)
