FixDVO is a marketplace platform design created in Figma to connect construction workers with clients needing repair and maintenance services. The interactive prototype demonstrates the user interface and workflows for job discovery, worker management, and service transactions.

## Design Overview

This is a **design prototype** showcasing the user interface and interactions for a construction services marketplace. The prototype includes:
- Job listing and discovery screens
- Worker profile management interfaces
- Service request workflows
- Messaging and communication screens
- Rating and review displays
